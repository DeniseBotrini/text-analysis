# Text Analysis Project

## Descrizione
Questo progetto ha l'obiettivo di analizzare testi in lingua italiana per estrarre informazioni significative, identificare 
pattern linguistici e generare insight utili nel cambiamento degli spot pubblicitari dagli anni '80 ad oggi. 

## Caratteristiche principali
- **Pulizia del testo**: rimozione di stopwords, punteggiatura e caratteri speciali.
- **Tokenizzazione e lemmatizzazione**: preparazione dei dati testuali per l'analisi.
- **Analisi statistica**: frequenza di parole, n-gram, distribuzione delle parole chiave.
- **Visualizzazione dei dati**: grafici a barre, word clouds e altri strumenti visuali.

## Tecnologie e librerie utilizzate
- Python 3.12
- Pandas
- Numpy
- NLTK / SpaCy / LDA
- Scikit-learn
